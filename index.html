常见问题

问题与答案

如何下载butterfly主题?

稳定分支[推荐]：git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly
开发分支：git clone -b dev https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

此主题对于部署的限制有哪方面?

此主题为全静态由node.js生成，可部署在Github、Vercel、Gitee、Coding、gitlab、fast、netlify也可直接部署在各大服务器厂商：阿里云、腾讯云等

如何成为致敬者成员？

对于Butterfly主题有二次开发魔改突出贡献的技术人员可成为致敬者成员
